# Advanced Routing

## Header-Based Routing

Route traffic based on HTTP headers. Useful for:
- A/B testing
- Feature flags
- User agent routing
- Custom header routing

## Query Parameter Routing

Route traffic based on URL query parameters.

## Method-Based Routing

Route traffic based on HTTP methods (GET, POST, PUT, DELETE, etc.).

## Multiple Match Conditions

Combine multiple conditions (path + header + method) for precise routing.

## Request Redirects

Redirect requests to different URLs or paths.

## URL Rewrites

Rewrite URLs before forwarding to backend services.
